# Draftpad

A multi-user note-taking app with block-based editing and cross-device sync.

## Tech Stack

**Frontend:** React (Vite), CSS Modules, Radix UI primitives
**Backend:** Node.js, Express
**Database:** PostgreSQL (Neon), Drizzle ORM
**Auth:** Hand-rolled server-side sessions
**Validation:** Zod
**Testing:** Vitest
**CI:** GitHub Actions
